Bagels Game

The computer game Bagels was created by Resek and Rowe from the Lawrence Hall of Science in Berkeley, California. In the game of Bagels, you must guess the number that the computer is thinking of. The computer generates a three digit number where each digit must be unique. You are provided with 20 turns to guess the number. The computer can give the following hints in response to each guess:

Pico - One digit correct but in the wrong position.
Fermi - One digit correct and in the right position.
Bagles - No digits correct.
